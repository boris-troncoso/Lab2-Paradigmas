Breve explicación del código del lab 2:

El proyecto se descompone en los siguientes archivos, los cuales se explican brevemente:

tda_card.pl: Cartas y ataques.

tda_deck.pl: Lógica de mazos y validaciones.

tda_juego.pl: Estado de jugadores y mesa.

main_195598568_BORIS_TRONCOSO_CONOPAN.pl: Motor principal con las reglas.

test.pl: Script para probar el funcionamiento.


Instrucciones para ejecutar el código:

1. Tener instalado SWI-Prolog.

2. Descargar el proyecto y desde la carpeta del lab 2 abrir una terminal y luego escribir: swipl

3. Luego le decimos cual es el archivo que deseamos que se lea, en este caso seria: [test].

4. Finalmente lanzamos la partida de test escribiendo: prueba_partida.

El programa mostrará en consola el flujo del juego y el resultado final.
